# UPN ESDAT — Estructura de Datos

Repositorio del curso **Estructura de Datos** (SIST1201A) de la Universidad Privada del Norte, correspondiente al ciclo 4, período 2026-1.

Este repositorio contiene materiales, guías y ejercicios desarrollados en C# con Visual Studio 2022, enfocados en la implementación de estructuras de datos dinámicas como listas, pilas, colas, árboles y grafos.

## Contenido del repositorio

### Unidad I — Listas, pilas y colas
- Semana 1: Introducción al curso, protocolo de seguridad y listas enlazadas simples.
- Semana 2: Operaciones sobre listas simples, ordenamiento, búsqueda y mezcla.
- Semana 3: Listas doblemente enlazadas y listas circulares.
- Semana 4: Análisis de problemas complejos.
- Semana 5: Pilas y evaluación de expresiones.
- Semana 6: Evaluación T1.
- Semana 7: Colas.

### Unidad II — Árboles y grafos
- Semana 8: Árboles binarios y colas avanzadas.
- Semana 9: Recorridos de árboles y ABB.
- Semana 10: Evaluación T2.
- Semana 11: Aplicaciones de árboles binarios.
- Semana 12: Grafos y representaciones.
- Semana 13: Evaluación T3.
- Semana 14: Caminos mínimos.
- Semana 15: Árbol de expansión mínima.
- Semana 16: Evaluación final.

## Requisitos

- Visual Studio 2022.
- .NET Framework 4.7.2 o superior.

## Cómo ejecutar

1. Clonar el repositorio.
2. Abrir la solución `.sln` del ejercicio o semana que quieras ejecutar.
3. Presionar **F5** en Visual Studio.
