# UPN ESDAT — Estructura de Datos

Material y ejercicios del curso **Estructura de Datos** (SIST1201A)
Universidad Privada del Norte · Ingeniería de Sistemas Computacionales · Ciclo 4
Sede Los Olivos · Período 2026-1

Implementaciones en C# con Visual Studio 2022, sin uso de colecciones genéricas. El propósito del curso es construir las estructuras de datos desde cero para entender el manejo directo de punteros y nodos.

## Requisitos

- Visual Studio 2022
- .NET Framework 4.7.2 o superior

## Cómo ejecutar

1. Clonar el repositorio
2. Abrir el archivo `.sln` del ejercicio que quieras ejecutar
3. Presionar **F5** en Visual Studio
